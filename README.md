# High-Order epistasis

Companion Github to the paper "On inferring epistatic complexity in protein sequence-function landscapes". 

The code is divided in 3 notebooks: 
- `epistasis_CR9114.ipynb` runs the RBA/RFA analysis on the CR9114 dataset (very slow and require a lot of memory). 
- `Figure_data_generation.ipynb` produce the figures (except the last two)
- `Global epistasis.ipynb`  generate the data and figures for the discussion on global epistasis 
- `Beyond_r2.ipynb` generate the data and figures for the discussion on the impact of epistasis on evolutionary trajectories.
